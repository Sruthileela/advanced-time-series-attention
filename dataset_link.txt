# Dataset Information
## Primary Dataset
Beijing Multi-Site Air-Quality Dataset
- Source: UCI Machine Learning Repository
- Link: https://archive.ics.uci.edu/ml/datasets/Beijing+Multi-Site+Air-Quality+Data
- Description: Hourly air quality data from 12 stations in Beijing (2013-2017)
- Features: PM2.5, PM10, SO2, NO2, CO, O3, temperature, pressure, dew point, etc.
- Size: ~420,000 records, 18 features per station

## Alternative Datasets

3. Synthetic Data Generator:

## Data Preparation
1. Download data.csv from UCI link
2. Place in data/raw/ folder
3. Ensure file is named 'air_quality.csv' or update config.yaml
4. Run preprocessing: python -c "from src.data_preprocessing import load_and_preprocess; df = load_and_preprocess('data/raw/air_quality.csv')"
