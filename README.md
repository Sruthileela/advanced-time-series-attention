# Advanced Time Series Forecasting using LSTM with Attention

## Overview
This project implements an advanced deep learning model for time series forecasting using LSTM enhanced with an Attention mechanism. The model captures temporal dependencies and improves forecasting accuracy.

## Objectives
- Build an LSTM-based time series forecasting model
- Integrate attention mechanism
- Evaluate using MSE, RMSE, MAE

## Project Files
- `model.py` – Model implementation
- `train.py` – Training script
- `dataset_link.txt` – Dataset reference
- `results.txt` – Evaluation metrics
- `requirements.txt` – Dependencies

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the model: `python train.py`
3. Check results in `results.txt`

## Conclusion
Attention mechanism significantly improves time series forecasting accuracy.
