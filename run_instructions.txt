# Advanced Time Series Forecasting - Run Instructions

## Setup Environment
1. Clone repository:
   git clone <repository-url>
   cd <repository-folder>

2. Create virtual environment (optional but recommended):
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

## Prepare Data
4. Download dataset:
   - Place your dataset in data/raw/ folder
   - Default expects: data/raw/air_quality.csv
   - Or update config.yaml with your dataset path

## Training Models
5. Train LSTM model:
   python train.py --model lstm --epochs 100 --batch_size 32

6. Train Transformer model:
   python train.py --model transformer --epochs 100 --batch_size 32

7. With hyperparameter tuning:
   python train.py --model lstm --tune --trials 20

## Generate Predictions
8. Make predictions with LSTM:
   python model.py --model models/lstm_final.h5 --model_type lstm --predict

9. Make predictions with Transformer:
   python model.py --model models/transformer_final.h5 --model_type transformer --predict

## Explainability
10. Generate SHAP explanations:
    python model.py --model models/lstm_final.h5 --model_type lstm --explain

11. Generate SHAP for Transformer:
    python model.py --model models/transformer_final.h5 --model_type transformer --explain

## Notebooks (Optional)
12. Explore data and results in Jupyter notebooks:
    jupyter notebook notebooks/
    - Run notebooks in numerical order

## Expected Outputs
- Trained models in models/ folder
- Metrics in results.txt
- Visualizations in results/ folder
- SHAP plots in results/ folder

## Troubleshooting
- If SHAP installation fails on Windows, try:
  pip install shap --no-deps
  pip install numpy scipy scikit-learn pandas matplotlib

- If CUDA errors occur, ensure TensorFlow GPU version is installed
