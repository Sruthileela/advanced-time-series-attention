Project Execution Instructions:

1. Ensure Python version 3.8 or above is installed
2. Clone or download the project repository
3. Navigate to the project directory
4. Install required dependencies using:
   pip install -r requirements.txt
5. Ensure data.csv is present in the root directory
6. Run the training and evaluation script:
   python train.py
7. View RMSE and MAE results in the terminal
8. SHAP explainability plot will be displayed automatically
